# sql-business-analytics

Projetos educacionais em SQL voltados para análise de dados, manipulação de bancos relacionais, construção de indicadores e desenvolvimento de soluções analíticas aplicadas a cenários de negócio.

## Projetos

### 🔗 Dashboard de Acompanhamento de Vendas

Projeto desenvolvido para análise de desempenho comercial utilizando SQL, explorando métricas de conversão, comportamento dos usuários e indicadores de vendas.

**Principais análises realizadas:**

- Receita mensal
- Quantidade de leads
- Quantidade de vendas
- Taxa de conversão
- Ticket médio
- Estados com maior volume de vendas
- Marcas mais vendidas
- Lojas com melhor desempenho
- Dias da semana com maior número de visitas

**Principais técnicas utilizadas:**

- CTEs
- JOINs
- GROUP BY
- HAVING
- Funções agregadas
- Tratamento de datas
- CASE WHEN
- KPIs comerciais

**Consultas desenvolvidas:**

- Receita, leads e conversão mês a mês
- Estados que mais venderam
- Marcas com maior participação
- Ranking de lojas
- Comportamento de acesso ao site

### 🔗 Análise do Perfil dos Leads

Projeto de segmentação e análise exploratória com foco em identificação de padrões demográficos, profissionais e comportamentais dos clientes.

**Principais análises realizadas:**

- Distribuição por gênero
- Status profissional
- Faixa etária
- Faixa salarial
- Classificação dos veículos visitados
- Idade dos veículos
- Marcas mais acessadas
- Modelos com maior interesse

**Principais técnicas utilizadas:**

- CTEs
- CASE WHEN
- GROUP BY
- Funções agregadas
- Tratamento de dados
- Tratamento de datas
- Regras de negócio
- Segmentação analítica

**Consultas desenvolvidas:**

- Gênero dos leads
- Perfil profissional
- Distribuição salarial
- Faixas etárias
- Classificação de veículos
- Idade média dos veículos
- Veículos mais visitados por marca
